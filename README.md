# node-task-todo 一个简易的任务清单管理工具 

# 安装
`
npm i -g  wly-task-todo
`
# 使用

`
task -h   // 展示所有命令用法
`

· Mac 需要将cli加入可执行文件当中

`
chmod +x  cli.js
`
